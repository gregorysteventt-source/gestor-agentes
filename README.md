# Gestor de Guardias

Version preparada para publicar en GitHub Pages.

Archivo principal:
- `index.html`

Notas:
- La app sigue usando Supabase para iniciar sesion y guardar datos.
- El repositorio puede ser publico porque el acceso real a los datos queda protegido por Supabase.
- Despues de publicar, agrega la URL de GitHub Pages en Supabase:
  - Authentication > URL Configuration > Site URL
  - Authentication > URL Configuration > Redirect URLs
